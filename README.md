## 代码规范
1. 类名: UpperCamelCase
2. 方法名: lowerCamelCase
3. 常量: CONSTANT_CASE
4. 非常量: lowerCamelCase
5. 参数: lowerCamelCase
6. 局部变量: lowerCamelCase
7. `@Override`: 合法则标注
8. 变量声明: 不允许组合声明
9. 代码格式化: IDEA格式
10. `if, else, for, do, while`: 必须加`{}`
11. 注释: 必须包含作者
